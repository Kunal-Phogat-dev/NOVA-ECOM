# NOVA — Cyber Luxury Fashion E-commerce

![NOVA Hero](https://via.placeholder.com/1200x600/0a0a0a/00f0ff?text=NOVA+SPRING+24)

**NOVA** is a high-end, futuristic fashion e-commerce experience featuring a sleek dark theme, vibrant cyan accents, and immersive editorial design.

## ✨ Features

### Must-Have
- **Stunning Homepage** with full-bleed cinematic hero
- **Premium Product Detail Pages** with image gallery, size & color selectors
- **Smooth Cart Drawer** with Framer Motion animations
- **Fully Responsive** design (mobile-first)
- **Checkout Flow** with Stripe integration
- **Collection & Category Pages** with sorting

### Advanced
- **Search Modal** with autocomplete
- **Wishlist** (Supabase authenticated)
- **User Authentication** (Supabase Auth)
- **Advanced Filters** & Sorting
- **Loading Skeletons** & micro-interactions

## 🎨 Design
- Deep black background (`#0a0a0a`)
- Signature cyan accent (`#00f0ff`)
- All-caps luxury typography
- Cinematic product photography
- Minimal, editorial aesthetic

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Backend**: Supabase (Auth + Database)
- **Payments**: Stripe
- **Icons**: Lucide React
